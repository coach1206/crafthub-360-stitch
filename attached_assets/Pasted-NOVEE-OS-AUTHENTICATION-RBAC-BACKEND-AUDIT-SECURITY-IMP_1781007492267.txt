FOUNDER_LEVEL_0

Founder Level 0 is supreme authority.

No role may:

* Modify Founder account
* Delete Founder account
* Transfer ownership
* Create another Founder account
* Change Founder permissions
* Override Founder restrictions
* Change production secrets
* Change payment controls
* Delete audit logs

Founder Level 0 only:

* Ownership transfer approval
* Developer approval
* Payment configuration approval
* Security override approval
* System lock controls

Login:

/founder-login

Requirements:

* Email
* PIN
* Founder Challenge Secret

JWT Lifetime:

2 Hours

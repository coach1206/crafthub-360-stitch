# Phase 7 Design Reference Pack

This folder collects the Phase 7 redesign reference images that currently
exist in the repo. It was created by copying images already present flat
under `public/` into a dedicated, discoverable location for the redesign
work. These are **guidance only** — Phase 7A uses them for layout
hierarchy, mood, texture, navigation pattern, and interaction styling
direction, not for 1:1 visual copying.

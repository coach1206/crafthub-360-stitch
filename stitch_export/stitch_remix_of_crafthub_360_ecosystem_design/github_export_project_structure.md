# CraftHub 360 Ecosystem - Frontend Export Structure

## 📂 Root Directory
- `index.html` - Main entry point with NOVEE OS boot sequence.
- `package.json` - Frontend dependencies and scripts.
- `tailwind.config.js` - Full design system token mapping.
- `README.md` - Deployment and architecture overview.

## 📂 /src
### 📂 /components
- `Navigation/` - TopAppBar, BottomNavBar, NavigationDrawer (Tactile variants).
- `Coaching/` - AI Help Coach overlays and Voice Interaction modules.
- `Cards/` - Premium glass-morphism cards for Inventory, Passport, and Craft Modules.
- `Controls/` - Haptic-ready buttons, sliders, and toggle switches.

### 📂 /screens
- `GuestJourney/` - NOVEE OS Power-Up, SmokeCraft, PourCraft, BeerCraft, WineCraft flows.
- `Passport/` - 360 Passport Connection, QR Scan, Networking Stamps.
- `Staff/` - POS 3 Terminal, Live Venue Map, Training Center.
- `Admin/` - E.A.T. Command Center, Inventory Vault, Venue Control Engine.

### 📂 /styles
- `main.css` - Global Tailwind imports and custom glass-morphism utilities.
- `animations.css` - Cinematic transitions and haptic pulse definitions.

### 📂 /assets
- `logos/` - CraftHub 360 Gold Foil Logo.
- `images/` - High-fidelity product and venue photography.
